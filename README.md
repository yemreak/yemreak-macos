# Voice first clipboard for your AI

Copy while talking. It goes to the right place.

## How it works

`⌥A` → Start talking → Copy anything → It inserts at the right timestamp → `⌥A` → Done

## Requirements

- macOS 14.0+
- Apple Silicon (M1/M2/M3)

## ...and more

- Offline dictation
- Session tracking
- Extra brightness
- Keyboard cleaner
