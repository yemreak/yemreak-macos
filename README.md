## Features

- **Dictation** - Voice to text, clipboard mode, choose your language
- **Display** - Go brighter than normal, control with F1/F2
- **Session** - Track your screen time, get reminded to take breaks
- **Tools** - Lock your keyboard to clean it

### Dictation

- **Clipboard Mode** `Option+A` - Include what you copied for AI conversations
- **Voice Dictation** `Option+Shift+A` - Voice to text, paste directly

- **Session Tracking** - See how long you've been at your computer
- **Idle Alerts** - Get reminded when you've been inactive too long

## Installation

1. Download `yemreak-macOS.dmg` from [Releases](https://github.com/yemreak/yemreak-macos/releases)
2. Drag to Applications
3. Grant permissions: Microphone, Accessibility

## Requirements

- macOS 14.0+
- Apple Silicon (M1/M2/M3)

## Updates

Menu Bar -> Check for Updates...

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Option+A` | Clipboard mode |
| `Option+Shift+A` | Voice dictation |
| `Option+Escape` | Cancel transcription |
| `Escape` | Cancel recording |
| `F1/F2` | Brightness down/up |
